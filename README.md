# ClickerGame

-   Jako téma pro tuto hru jsem vybral farmaření.

-   Přijde mi to jako ideální nápad, protože v těchto hrách je rychle
    vidět progress, a uživatel se rychle bude cítit úspěšně, že se mu
    něco povedlo.

-   Hlavním cílem hry bude rozšiřovat svoje pole, pěstovat na nich nové
    suroviny, které se pak budou prodávat a na nich vydělat peníze.

-   Zajímavé prvky na hře jsou:

    -   **Princip rozšiřování polí:**

        -   Hráč začne s jedním, tedy může pěstovat jen jednu věc.
            Postupně ale může kupovat další a tím pádem exponenciálně
            zvětšit svůj zisk

    -   **Prodávání na trhu:**

        -   Hráč si vybere, kam prodá svoje suroviny, podle poptávky
            (hodně peněz ale hodně času / málo peněz ale málo času)

    -   Hráč klikáním zrychluje čas, tedy i růst surovin

    -   **Upgrady: Možnost pomocníků**

        -   Možnost koupit např. traktor, který urychlí růst surovin,
            nebo pomocníka, který automaticky prodá suroviny
